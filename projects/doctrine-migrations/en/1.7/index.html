Migrations Documentation

The Doctrine Migrations documentation is a reference guide to everything you need to know about the migrations project.

Getting Help

If this documentation is not helping to answer questions you have about Doctrine Migrations don't panic. You can get help from different sources:

  • The Doctrine Mailing List
  • Gitter chat room #doctrine/migrations
  • Report a bug on GitHub.
  • On StackOverflow

The best way to get started is with the Introduction section. Use the sidebar to browse other documentation for the Doctrine PHP Migrations project.
