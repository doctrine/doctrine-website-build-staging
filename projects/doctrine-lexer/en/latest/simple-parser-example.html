Simple Parser Example

Extend the Doctrine\Common\Lexer\AbstractLexer class and implement the getCatchablePatterns, getNonCatchablePatterns, and getType methods. Here is a very simple example lexer implementation named CharacterTypeLexer. It tokenizes a string to T_UPPER, T_LOWER andT_NUMBER tokens:

1<?php

2

3use Doctrine\Common\Lexer\AbstractLexer;

4

5class CharacterTypeLexer extends AbstractLexer

6{

7    const T_UPPER =  1;

8    const T_LOWER =  2;

9    const T_NUMBER = 3;

10

11    protected function getCatchablePatterns()

12    {

13        return array(

14            '[a-bA-Z0-9]',

15        );

16    }

17

18    protected function getNonCatchablePatterns()

19    {

20        return array();

21    }

22

23    protected function getType(&$value)

24    {

25        if (is_numeric($value)) {

26            return self::T_NUMBER;

27        }

28

29        if (strtoupper($value) === $value) {

30            return self::T_UPPER;

31        }

32

33        if (strtolower($value) === $value) {

34            return self::T_LOWER;

35        }

36    }

37}

Use CharacterTypeLexer to extract an array of upper case characters:

1<?php

2

3class UpperCaseCharacterExtracter

4{

5    private $lexer;

6

7    public function __construct(CharacterTypeLexer $lexer)

8    {

9        $this->lexer = $lexer;

10    }

11

12    public function getUpperCaseCharacters($string)

13    {

14        $this->lexer->setInput($string);

15        $this->lexer->moveNext();

16

17        $upperCaseChars = array();

18        while (true) {

19            if (!$this->lexer->lookahead) {

20                break;

21            }

22

23            $this->lexer->moveNext();

24

25            if ($this->lexer->token['type'] === CharacterTypeLexer::T_UPPER) {

26                $upperCaseChars[] = $this->lexer->token['value'];

27            }

28        }

29

30        return $upperCaseChars;

31    }

32}

33

34$upperCaseCharacterExtractor = new UpperCaseCharacterExtracter(new CharacterTypeLexer());

35$upperCaseCharacters = $upperCaseCharacterExtractor->getUpperCaseCharacters('1aBcdEfgHiJ12');

36

37print_r($upperCaseCharacters);

The variable $upperCaseCharacters contains all of the upper case characters:

1Array

2(

3    [0] => B

4    [1] => E

5    [2] => H

6    [3] => J

7)

This is a simple example but it should demonstrate the low level API that can be used to build more complex parsers.
