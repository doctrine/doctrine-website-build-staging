protected string $configuration Set the configuration key for the Configuration. Configuration key is assembled as "doctrine.configuration.{key}" and pulled from service locator.
protected string $eventmanager Set the eventmanager key for the EventManager. EventManager key is assembled as "doctrine.eventmanager.{key}" and pulled from service locator.
protected null|string|PDO $pdo Set the PDO instance, if any, to use. If a string is set then the alias is pulled from the service locator.
protected string $driverClass Setting the driver is deprecated. You should set the driver class directly instead.
