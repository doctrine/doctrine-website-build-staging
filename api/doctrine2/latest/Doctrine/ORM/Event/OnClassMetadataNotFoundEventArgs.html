OnClassMetadataNotFoundEventArgs

class OnClassMetadataNotFoundEventArgs extends ManagerEventArgs (View source)

Class that holds event arguments for a onClassMetadataNotFound event.

This object is mutable by design, allowing callbacks having access to it to set the found metadata in it, and therefore "cancelling" a onClassMetadataNotFound event
