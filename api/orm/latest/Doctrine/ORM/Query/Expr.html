Creates a conjunction of the given boolean expressions.

Example:

[php]
// (u.type = ?1) AND (u.role = ?2)
$expr->andX($expr->eq('u.type', ':1'), $expr->eq('u.role', ':2'));

Parameters

Comparison|Func|Orx|string $x Optional clause. Defaults to null, but requires at least one defined when converting to string.
