string walkSelectStatement(SelectStatement $AST)

Replaces the whereClause in the AST.

Generates a clause equivalent to WHERE IN (:dpid_1, :dpid_2, ...)

The parameter namespace (dpid) is defined by the PAGINATOR_ID_ALIAS

The total number of parameters is retrieved from the HINT_PAGINATOR_ID_COUNT query hint.

SelectStatement $AST
