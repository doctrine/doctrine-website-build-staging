SQLServer2012Platform

class SQLServer2012Platform extends SQLServer2008Platform

Platform to ensure compatibility of Doctrine with Microsoft SQL Server 2012 version.

Differences to SQL Server 2008 and before are that sequences are introduced, and support for the new OFFSET... FETCH syntax for result pagination has been added.
